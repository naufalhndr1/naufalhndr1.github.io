Wellcome To My Website

Hallo nama saya Naufal Hendrawan atau bisa dipanggil Opal. Saya lahir di Sukabumi pada tanggal 9 Agustus 2000. Di website ini saya akan menceritakan tentang hobi saya.

Hobi saya adalah bermusik, saya sangat suka bermain alat musik seperti gitar dan piano selain itu saya juga suka sekali benyanyi. Hobi saya yang lainnya adalah menulis, saya senang sekali menulis karena lewat menulis saya dapat menuangkan seluruh ide dan hal - hal baik lewat menulis. Hobi terakhir saya adalah berkelana, maksud saya berkelana dalam artian menjelajah keindahan dari indonesia termasuk yang paling penting kota kelahiran saya Sukabumi, saya sangat senang menjelajahi puncak - puncak tertinggi di Indonesia mendaki gunung maupun terjun menjelajahi alam bawah laut Indonesia
